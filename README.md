## Radioactive Decay Calculator

Steup:
    1. Build a Python environment with numpy matplotlib.
    2. Change the Python library path(/home/hridoy/miniconda3/lib/python3.13/site-packages/numpy/_core/include) in [CMakeLists.txt](CMakeLists.txt) with your system Python library path.
    3. Install CMake in your system.(Optional)

Run Program: 
    1. Go to build directory.
    2. open the directory in terminal
    3. write ./Radioactive_deacy_calculator
    4. Now you can get results as shown in the picture below:
![Process.png](Process.png)
![decay.png](build/decay.png)

Result:
    1. The final result will show in the terminal and
    2. The plot will be saved in (build/decay.png) if you want to save.
